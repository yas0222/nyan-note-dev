# にゃん・ノート

猫の健康記録を日々残すアプリです。

## 目的

- 毎日のごはん・飲水・排泄・体重などを記録し、猫の体調変化を継続的に把握する。

## 現在できること

- 猫プロフィール登録・編集・削除
- 日次記録の保存・編集・削除
- メモ欄
- 記録カレンダー
- `localStorage` 保存
- Firestore 保存
- Firebase 匿名認証
- `ownerUid` による本人データ管理
- Firestore セキュリティルール前提の設計
- JSON バックアップ書き出し
- `publicCats` による公開プロフィール分離
- みんな画面で公開プロフィール一覧表示
- 都道府県フィルター
- 地域公開範囲: 非公開 / 都道府県まで / 市区町村まで

## データ方針

- `cats` / `records` は本人用
- `publicCats` は公開用
- `records` は公開しない
- 画像データは Firestore に保存しない
- 地域非公開の場合、`publicCats` には `prefecture` / `city` を保存しない

## 今後の候補

- 自分の猫をみんな画面に表示する/しない切り替え
- 市区町村フィルター
- 統計画面
- Google ログイン / Apple ログイン
- バックアップ復元

## ローカル実行

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。
