# にゃん・ノート

猫の健康記録を日々残すアプリです。

## v0.1 安定版

現在の実装を **v0.1 安定版** として区切ります。

v0.1 では以下が安定動作します。

- 猫プロフィール
- 日次記録
- メモ
- 記録カレンダー
- Firestore 保存
- Firebase 匿名認証
- JSON バックアップ
- `publicCats` によるみんな画面

## 目的

- 毎日のごはん・飲水・排泄・体重などを記録し、猫の体調変化を継続的に把握する。

## 現在できること

- 猫プロフィール登録・編集・削除
- 日次記録の保存・編集・削除
- メモ欄
- 記録カレンダー
- Firestore 保存
- Firebase 匿名認証
- JSON バックアップ書き出し
- `publicCats` による公開プロフィール分離
- みんな画面で公開プロフィール一覧表示

## データ方針

- `cats` / `records` は本人用データ
- `publicCats` は公開用プロフィール
- `records` は公開しない
- 地域非公開の場合、`publicCats` には都道府県・市区町村を保存しない
