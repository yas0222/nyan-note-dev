# 猫の健康共有アプリ（MVPプロトタイプ）

このリポジトリは、設計書をもとにした **ローカルで動く簡易Webアプリの雛形** です。

初心者でも触りやすいように、技術構成は以下にしています。

- HTML
- CSS
- JavaScript（フレームワークなし）
- ブラウザの `localStorage` を簡易DBとして利用

## できること（MVP）

- 猫プロフィール登録
  - 名前、年齢、性別、都道府県、市区町村、公開設定、写真URL
- 日次健康記録
  - エサ総量、カリカリ/ウェット比率、おやつ量、うんち・おしっこ回数、メモ
  - 比率合計が100%かどうかのバリデーション付き
- 地域別統計
  - 地域（都道府県+市区町村）を選んで平均値を表示
  - 全国（すべて）も表示可能

## 画面

タブ切り替えで以下の3画面を実装しています。

1. 猫プロフィール登録
2. 日次健康記録
3. 地域別統計

## ローカル実行方法

`index.html` を直接開いても動作しますが、以下のように簡易サーバー起動を推奨します。

```bash
python3 -m http.server 8000
```

起動後、ブラウザで以下を開いてください。

- http://localhost:8000
